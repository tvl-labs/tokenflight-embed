title: Theming & Custom Colors
description: Customize the look of TokenFlight widgets with built-in themes, auto dark mode, and CSS variable overrides.
---

TokenFlight widgets ship with **light** and **dark** themes, plus an **auto** mode that follows the user's system preference. You can further override any color via `setCustomColors()` or CSS variables.

## Built-in Themes

### Declarative

```html
<tokenflight-swap theme="dark"></tokenflight-swap>
<tokenflight-swap theme="light"></tokenflight-swap>
<tokenflight-swap theme="auto"></tokenflight-swap>
```

### Imperative

```js
import { TokenFlightSwap } from '@tokenflight/swap';

const swap = new TokenFlightSwap({
  container: '#swap',
  config: { theme: 'auto' },
});
swap.initialize();
```

`auto` detects `prefers-color-scheme: dark` and picks accordingly.

### Switching at Runtime

```js
swap.setTheme('light');
swap.setTheme('dark');
swap.setTheme('auto');
```

No re-initialization needed — the widget updates instantly.

## Custom Color Overrides

Use `setCustomColors()` or the `customColors` config option to override any theme color.

### Via Config

```js
const swap = new TokenFlightSwap({
  container: '#swap',
  config: {
    theme: 'dark',
    customColors: {
      primary: '#FF6B00',
      background: '#1A1A2E',
    },
  },
});
swap.initialize();
```

### Via Runtime Method

```js
swap.setCustomColors({
  primary: '#FF6B00',
  background: '#1A1A2E',
  textPrimary: '#FFFFFF',
  border: '#2A2A4A',
});
```

### Named Color Keys

These friendly names map to internal CSS variables:

| Key | CSS Variable | Description |
|-----|-------------|-------------|
| `primary` | `--tf-accent` | Buttons, links, active states |
| `background` | `--tf-bg` | Main background |
| `textPrimary` | `--tf-text` | Primary text |
| `textSecondary` | `--tf-text-secondary` | Labels, hints |
| `border` | `--tf-border` | Borders, dividers |
| `success` | `--tf-success` | Success states |
| `error` | `--tf-error` | Error states |
| `warning` | `--tf-warning` | Warning states |

Any key not in the table above is converted to `--tf-{key}` automatically:

```js
swap.setCustomColors({ 'bg-secondary': '#111' });
// sets --tf-bg-secondary
```

## Custom Fonts

The widget does **not** bundle any font files — it uses system font stacks by default. You can override fonts with any typeface you load on your page.

### Font CSS Variables

| Variable | Default | Description |
|----------|---------|-------------|
| `--tf-font` | `'DM Sans', system-ui, -apple-system, sans-serif` | UI text (labels, buttons, headings) |
| `--tf-font-mono` | `'JetBrains Mono', ui-monospace, 'SFMono-Regular', monospace` | Numeric values (balances, amounts, addresses) |

### Loading Fonts & Applying

Load your preferred font (via Google Fonts, Fontsource, local files, etc.) and override the CSS variables:

```html
<!-- 1. Load the font -->
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap" rel="stylesheet">

<!-- 2. Override the widget font -->
<style>
  tokenflight-swap {
    --tf-font: 'Inter', sans-serif;
  }
</style>

<tokenflight-swap api-endpoint="..."></tokenflight-swap>
```

### Via Imperative API

Use `setCustomColors()` — it supports any CSS variable, including fonts:

```js
swap.setCustomColors({
  font: "'Inter', sans-serif",
  'font-mono': "'Fira Code', monospace",
});
```

### Using Self-Hosted Fonts (Fontsource)

If you want the original look with self-hosted fonts:

```bash
npm install @fontsource/dm-sans @fontsource/jetbrains-mono
```

```js
import '@fontsource/dm-sans/latin-400.css';
import '@fontsource/dm-sans/latin-500.css';
import '@fontsource/dm-sans/latin-600.css';
import '@fontsource/dm-sans/latin-700.css';
import '@fontsource/jetbrains-mono/latin-400.css';
import '@fontsource/jetbrains-mono/latin-500.css';
```

The widget will pick up DM Sans and JetBrains Mono from the default `--tf-font` / `--tf-font-mono` values automatically — no variable override needed.

### Using System Fonts Only

If you don't load DM Sans or JetBrains Mono, the widget gracefully falls back to system fonts (`system-ui`, `ui-monospace`, etc.) which look great on all platforms with zero network cost.

## Example: Brand-Colored Widget

```js
const swap = new TokenFlightSwap({
  container: '#swap',
  config: {
    theme: 'dark',
    customColors: {
      primary: '#E8396B',        // brand pink
      background: '#12121A',     // deep bg
      'bg-secondary': '#1A1A28',
      'surface': '#1E1E2E',
      border: '#2D2D44',
    },
  },
});
swap.initialize();
```

## Example: Embed on a Light Page

```js
swap.setCustomColors({
  primary: '#2563EB',          // blue
  background: '#FAFAFA',
  border: '#E5E7EB',
  'surface-hover': '#F3F4F6',
});
```
