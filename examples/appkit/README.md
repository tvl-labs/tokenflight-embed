# TokenFlight AppKit Example

## Run locally

```bash
pnpm install
pnpm --filter @tokenflight/swap build
pnpm --filter @tokenflight/adapter-appkit build
pnpm --filter @tokenflight/example-appkit dev
```

## Environment

Create `.env` in this folder:

```bash
VITE_REOWN_PROJECT_ID=your-walletconnect-project-id
```
